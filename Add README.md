# 🌸 TEMITOPE – Luxury Perfume Brand Website

TEMITOPE is a modern, elegant, multi-page perfume brand website designed to showcase premium fragrances through clean layouts, beautiful typography, and an immersive user experience.

This project focuses on creating a luxury aesthetic while maintaining responsive design and smooth navigation across multiple pages.
